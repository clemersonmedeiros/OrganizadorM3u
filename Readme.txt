# ORGANIZADOR DE ARQUIVOS M3U	

# --------------------------------------------------------------------- #
#			ORGANIZADOR DE ARQUIVOS M3U			#			
# --------------------------------------------------------------------- #

Desenvolvida em 01/02/2019

Essa aplicação permite que o usuário classifique sua lista de canais de um
arquivo M3U, permitindo assim gerar um arquivo novo nesse formato apenas com
os itens que foram selecionados.

Para executar:

- Abrá o index da aplicação, não é necessário instalar nada.

Características: 

- Quando carregado um arquivo, ele separa em categorias do mesmo nome;
- Quando clicado sobre um item único (canal, filme, equisódio,...) a aplicação
cria uma cópia do nome da categoria e replica para a outra lista adicionando o 
item selecionado apenas, ao clicar sobre outro item, ele vai categorizando de 
forma correta também.
- Ao arrastar uma categoria para a outra lista, ele adiciona todos os itens junto;
- Ao clicar no ícone de Download em uma das listas, ele irá baixa-lo;
